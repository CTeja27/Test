A modern management workspace for arts organizations.

AMS Desktop is inspired by AMS Network — built for organizations that want a clean, shared system without complex setup. One computer becomes the Host (runs services locally), and other computers join as Clients to work on the same data in real time.

How it works (Host + Client)

Set up once, then your whole team connects to the same workspace on your local network.

1

Choose Host or Client

Host runs the backend. Clients connect to the Host and share the same data.

HostRuns backend services ClientConnects to Host URL
2

Auto-find the Host on Wi-Fi

Clients can scan the LAN to detect the Host and auto-fill the server URL.

DiscoveryUDP 49501 broadcast FallbackManual Host URL entry
3

Services run on the Host

Rails API + database + cache + MediaConvert run locally on the Host.

APITCP 4000 MediaConvertTCP 5001 StoragePostgres + Redis
4

Firewall rules (Host)

Allow discovery + service ports on the Host firewall so Clients can connect.
